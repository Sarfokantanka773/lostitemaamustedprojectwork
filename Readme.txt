Download and Install any local web server such as XAMPP/WAMP.
Download the provided source code zip file. (download button is located below)

Installation

1. Open your XAMPP/WAMP's Control Panel and start the "Apache" and "MySQL".
2. Extract the source code zip file  "Lost Items AAMUSTED".
3. If you are using XAMPP, copy the extracted source code folder and paste it into the xampp's "htdocs" directory. If you are using WAMP, paste it inside the "www" directory.
Open a web browser and browse the PHPMyAdmin. (http://localhost/phpmyadmin)
Create a new database naming "lfis_db".
Import the provided SQL file into your newly created database. The SQL file is known as "lfis_db" and it is located inside the "SQL file" directory of the source code folder.
Browse the web application in a browser. i.e.(http://localhost/Lost Items AAMUSTED /) for the users' side and (http://localhost/Lost Items AAMUSTED/admin) for the admin side
